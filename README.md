# 🌐 Weatherly

App del clima en tiempo real para cualquier ciudad del mundo. Construida con React, Vite y Tailwind CSS. **Sin API key necesaria.**

## ✨ Funcionalidades

- Temperatura actual y sensación térmica
- Hora de amanecer y atardecer con indicador en tiempo real
- Humedad, presión atmosférica, visibilidad
- Velocidad y dirección del viento con mini brújula
- Índice UV con clasificación
- Precipitación actual
- Pronóstico por hora (próximas 24h)
- Pronóstico a 7 días
- Búsqueda de ciudades con autocompletado
- Fondo dinámico según hora del día y condición climática

## 🛠️ Stack

- React 19 + Vite 6
- Tailwind CSS 3
- Open-Meteo API (gratis, sin API key)

## 🚀 Local

```bash
npm install
npm run dev
```

## 🌍 Deploy en Vercel

```bash
npm install -g vercel
vercel
```

O importa el repo en vercel.com — detecta Vite automáticamente.
No se necesitan variables de entorno.
